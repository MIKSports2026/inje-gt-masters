WIREFRAME NOTES — 현재 vs 개선안

현재 (AS-IS)

  • 클래스 목록 단순 나열, 시각적 위계 없음
  • 참가비 정보 없음
  • 기술 스펙 없음
  • 탭 전환 없음 — 스크롤로만 탐색
  • 차량 이미지 미활용
  • REGISTER 버튼 연결 없음

개선안 (TO-BE)

  • 5개 클래스 탭 전환 — skewX(-15deg) 사선 스타일
  • 차량 풀블리드 배경 + 그라데이션 오버레이
  • 좌측: 클래스명(대형) + tagline + 참가비(라운드/연간)
  • 우측: TECHNICAL SPECS 5종 표 형식
  • REGISTER NOW 버튼 → /entry 직접 연결
  • Sanity 연동: 이미지·스펙·참가비 CMS 편집 가능
